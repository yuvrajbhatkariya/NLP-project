# 🛡️ Real-Time Fraud Detection System (Digital Arrest Protection)

## 🚨 Problem
Fraudsters use real-time calls to create fear, urgency, and pressure, tricking users into sharing sensitive information or money.  
Most systems detect fraud **after it happens**, not during the call.

---

## 💡 Solution
This project is a **real-time AI system** that listens to live conversations and detects fraud **while the call is happening**, helping users take immediate action.

---

## 🧠 How It Works
- Captures live audio from microphone  
- Detects speech segments using VAD  
- Converts speech to text  
- Maintains conversation context (memory + summary)  
- Uses an LLM to analyze fraud patterns  
- Outputs risk level, reason, and advice  

---

## 🎯 Key Idea
**Detect fraud before it happens, not after.**

---

## 🌍 Impact
- Prevents real-time scam attacks  
- Protects users from financial loss  
- Acts as an intelligent safety assistant
